# Supply-Chain-Nexus

An enterprise-grade smart supply chain and logistics dashboard prototype featuring a live inventory ledger, automated threshold monitoring, and real-time dispatch logging.

---

## 🚀 Key Features

*   **Live Inventory Ledger:** Real-time tracking of core commodities (Rice, Sugar, Oil) with automated status indicators (`HEALTHY`, `LOW`, `EMPTY`).
*   **AI Threshold Engine:** An intelligent monitoring layer that analyzes safety stock levels against predefined thresholds and automatically flags lines needing immediate procurement or reordering.
*   **Live Dispatch Desk:** A functional operational desk allowing operators to input target customer IDs, select commodities, and authorize instant dispatches.
*   **Real-Time Dispatch Log:** A chronological audit trail tracking every authorized dispatch with high-precision time stamping and quantity changes.

---

## 📊 Dashboard KPIs Tracked

| Metric | Description |
| :--- | :--- |
| **Total Stock Units** | Aggregate volume of all available inventory. |
| **Processed Orders** | Cumulative count of successfully authorized dispatches. |
| **Low Threshold Lines** | Active alerts for items dipping below safety margins. |
| **Out of Stock Items** | Critical alerts for items requiring immediate procurement. |

---

## 🛠️ Technology Stack

*   **Frontend:** HTML5, CSS3 (Enterprise Dark Theme), JavaScript (ES6+)
*   **State Management:** Real-time in-memory ledger and log arrays
*   **Logic Engine:** Threshold monitoring algorithms for automated supply chain recommendations
